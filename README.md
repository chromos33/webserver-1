WEBSERVER-PROJEKT TOOL

Das erste Tool ist ein Webserver Tool der die Web-Projekte aus Github zieht und diese auflistet, dann 
kann man zwischen dem Environment wählen  und 
anschließend die Domain setzen.

Das Tool bietet auch die Möglichkeit direkt die wichtigsten Aktionen mit dem Projekt zu starten wie 
z.B den Git Status, Übersicht über die infos des Projekt , die letzten 10 eingegangenen
Fehlermeldungen auf einem Blick, das Projekt direkt zu aktualisieren oder ein Composer Update.
